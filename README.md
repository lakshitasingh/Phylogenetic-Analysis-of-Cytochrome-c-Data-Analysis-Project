# Phylogenetic-Analysis-of-Cytochrome-c---Data-Analysis-Project
This project performs phylogenetic analysis of the Cytochrome c protein across multiple organisms to understand evolutionary relationships. Using bioinformatics and data analysis techniques, protein sequences from different species were analyzed to measure similarity, compute evolutionary distances, and construct a phylogenetic tree.

The project demonstrates how computational biology tools and Python-based data analysis can be used to study molecular evolution.
